## Deployment

### GitHub Pages
1. Push all files to your GitHub repo (e.g. `aiproject77/shininglegacy`)
2. Go to Settings > Pages > Source: Deploy from branch (main, / root)
3. Site will be live at `https://aiproject77.github.io/shininglegacy`

### Vercel (recommended — custom domain)
1. Connect GitHub repo to Vercel
2. No build config needed (static site)
3. Add custom domain `shininglegacy.ca` in Vercel project settings
4. Update DNS: add CNAME record pointing to `cname.vercel-dns.com`

### Connect to Google Business Profile
1. Go to https://business.google.com
2. Find your profile: "Shining Legacy Commercial Cleaning Inc"
3. Under "Info" > "Website" — enter `https://shininglegacy.ca`
4. Under "Advanced settings" > verify domain ownership via Google Search Console
5. The LocalBusiness schema in index.html is already configured with your business details
